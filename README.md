# National Family Health Survey (NFHS) Data Analysis

## Project Overview

This project analyzes data from the **National Family Health Survey (NFHS)** using Python to explore key health and demographic indicators across Indian states. The objective is to transform raw survey data into meaningful insights through data cleaning, exploratory data analysis (EDA), statistical analysis, and visualizations.

The analysis focuses on identifying patterns and relationships among indicators such as female literacy, child nutrition, vaccination coverage, maternal healthcare, institutional births, and anaemia among women.

---

## Objectives

- Analyze state-wise health and demographic indicators.
- Perform data cleaning and preprocessing.
- Conduct Exploratory Data Analysis (EDA).
- Compare health indicators using GroupBy and Pivot Table analysis.
- Study relationships between variables using correlation analysis.
- Create informative visualizations for better understanding.
- Extract meaningful insights to support data-driven decision-making.

## Key Insights

- Higher female literacy is associated with improved maternal healthcare and vaccination coverage.
- Child stunting varies significantly across Indian states.
- Institutional births are generally higher in states with stronger healthcare infrastructure.
- Anaemia among women remains a major public health concern.
- Correlation analysis reveals meaningful relationships among literacy, vaccination, maternal care, and child health indicators.
- Visualizations effectively highlight regional disparities and healthcare trends.
